# Create Test Document Using Google Docs (Free!)

## If You Don't Have Microsoft Word

### Step 1: Go to Google Docs
https://docs.google.com

### Step 2: Create New Document

### Step 3: Paste This Content:

```
Technical Report

This document contains sample text to test formatting validation.

Background Information

The background section provides context. This should be in Arial 12pt.

Methodology

This section describes the methods used in the study.

Results and Findings

The results showed interesting patterns in the data analysis.

Discussion

Further research is needed to validate these findings.
```

### Step 4: Apply Formats

**Format the headings:**
1. Select "Technical Report"
   - Format → Text → Size → 16
   - Format → Text → Times New Roman
   - Format → Paragraph styles → Heading 1

2. Select "Background Information"
   - Format → Text → Size → 14 (❌ Wrong!)
   - Format → Text → Calibri (❌ Wrong!)
   - Format → Paragraph styles → Heading 2

3. Select "Methodology"
   - Format → Text → Size → 16
   - Format → Text → Times New Roman
   - Format → Paragraph styles → Heading 2

**Format the paragraphs:**
1. All paragraph text: Arial, 12pt
   (Except make one paragraph Verdana or Calibri to create errors)

### Step 5: Download as DOCX
- File → Download → Microsoft Word (.docx)

### Step 6: Upload to Your App!

**Expected Errors:**
- Heading font mismatch
- Heading size mismatch
- Body font mismatch (if you changed one)
