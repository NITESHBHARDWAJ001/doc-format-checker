# 30-Second Test Document

## Fastest Way to Test

### Step 1: Open Microsoft Word

### Step 2: Type this:

```
Sample Report

This is a test paragraph.

Another Section

This is another paragraph.
```

### Step 3: Apply Formats

**Select "Sample Report":**
- Right-click → Font → Times New Roman, 16pt
- Click "Heading 1" style

**Select "This is a test paragraph":**
- Right-click → Font → Arial, 12pt

**Select "Another Section":**
- Right-click → Font → **Comic Sans MS**, **18pt** (intentional errors!)
- Click "Heading 2" style

**Select "This is another paragraph":**
- Right-click → Font → **Calibri**, **14pt** (intentional errors!)

### Step 4: Save as .docx

### Step 5: Upload to App

Set rules:
- Heading Font: Times New Roman
- Heading Size: 16
- Body Font: Arial
- Body Size: 12

**You should see 4 errors detected!**
